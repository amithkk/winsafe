Exposes item methods of a ListView control.

You cannot directly instantiate this object, it is created internally by the control.

Appends a new item by sending an LVM_INSERTITEM message, and returns its index.

The texts are relative to each column.

pub fn text(&self, item_index: u32, column_index: u32, buf: &mut WString)[src]

Retrieves the text of an item under a column by sending an LVM_GETITEMTEXT message.

The passed buffer will be automatically allocated.

This method can be more performant than text_str because the buffer can be reused, avoiding multiple allocations. However, it has the inconvenient of the manual conversion from WString to String.

Examples

ⓘ
use winsafe::{gui, WString};

let my_list: gui::ListView; // initialized somewhere

let mut buf = WString::default();
my_list.items().text(0, 2, &mut buf); // 1st item, 3rd column

println!("Text: {}", buf.to_string());

pub fn text_str(&self, item_index: u32, column_index: u32) -> String[src]

A more convenient text, which directly returns a String instead of requiring an external buffer.
