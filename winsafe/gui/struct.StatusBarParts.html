Exposes the part methods of a StatusBar control.

You cannot directly instantiate this object, it is created internally by the control.

pub fn text(&self, part_index: u8, buf: &mut WString)[src]

Retrieves the text of the item by sending a SB_GETTEXT message.

The passed buffer will be automatically allocated.

This method can be more performant than text_str because the buffer can be reused, avoiding multiple allocations. However, it has the inconvenient of the manual conversion from WString to String.

Examples

ⓘ
use winsafe::{gui, WString};

let my_sb: gui::StatusBar; // initialized somewhere

let mut buf = WString::default();
my_sb.parts().text(0, &mut buf).unwrap();

println!("Text: {}", buf.to_string());
