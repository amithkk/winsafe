pub fn CommDlgExtendedError() -> ERROR

CommDlgExtendedError function.

Note: The co::ERROR returned by this function cannot be properly formatted by Debug and Display traits, thus showing a wrong description. The formatted text actually will belong to the standard error code of the same value. Do not rely on these descriptions.
