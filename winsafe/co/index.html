Module winsafe::co[−][src]

Expand description

Native Win32 constants and types of constants.

All types can be converted from/to their underlying integer type. Some types implement the NativeBitflag trait and bitflag operations.

Among these constant types, three are error types: CDERR, ERROR and HRESULT.
