#[repr(C)]pub struct ERROR(_);

A Windows system error code (u32) retrieved by GetLastError function, or an HRESULT.

Implements the Error trait.

Implements the Debug and Display traits to show the error code along with the error description, taken from FormatMessage. For example, the code below:

ⓘ
use winsafe::co::ERROR;

println!("{}", ERROR::OUT_OF_PAPER);

Will print:

[0x001c 28] The printer is out of paper.
